99 skills

Every hs:* skill is prose guidance — a structured prompt that tells Claude Code what to do. The real blocking happens in hooks and scripts. Skills can be composed (chained subagents) or invoked standalone.Mỗi skill hs:* là hướng dẫn prose — một prompt có cấu trúc cho Claude Code biết cần làm gì. Việc chặn thực sự xảy ra trong hook và script. Skill có thể kết hợp (subagent nối) hoặc dùng độc lập.

💡
A skill is only as strong as the hook that enforces it. Browse the catalog to find the right skill — then see the rig to understand what actually blocks bad output from shipping.Skill chỉ mạnh bằng hook thực thi nó. Duyệt catalog để tìm skill phù hợp — rồi xem the rig để hiểu cái gì thực sự chặn đầu ra xấu khỏi ship.

🏷️
Invoke any skill with /hs:<name>. The badge on each card is its enforcement tier: workflow has gates & artifacts (can block the pipeline) · knowledge is advisory (generates output, never blocks) · gate writes a hard verdict the pipeline reads. Most skills are knowledge or workflow; only a few write a hard gate verdict. Gọi skill bằng /hs:<tên>. Badge trên mỗi thẻ là hạng cưỡng chế: workflow có gate & artifact (chặn pipeline được) · knowledge là tư vấn (tạo output, không chặn) · gate ghi verdict cứng pipeline đọc. Phần lớn skill là knowledge hoặc workflow; chỉ một số ít ghi verdict gate cứng.

FAQHỏi đáp

Are skills the thing that blocks bad output?Skill có phải thứ chặn đầu ra xấu?
No. A skill is prose — a structured prompt. Blocking happens in hooks and scripts (gate_stage, write_guard, the pre-push backstop). A skill is only as strong as the hook that enforces it.Không. Skill là prose — một prompt có cấu trúc. Việc chặn nằm ở hook và script (gate_stage, write_guard, backstop pre-push). Skill chỉ mạnh bằng hook thực thi nó.
How do I invoke a skill?Gọi một skill thế nào?
Type /hs:<name> in Claude Code — e.g. /hs:plan. Skills can be chained (orchestrators spawn subagents) or invoked standalone.Gõ /hs:<tên> trong Claude Code — ví dụ /hs:plan. Skill có thể nối (orchestrator sinh subagent) hoặc dùng độc lập.
